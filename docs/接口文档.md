# 翔泳社管理系统接口文档（简版）

## 通用说明
- Base URL：`http://localhost:8080`
- 认证：JWT（`Authorization: Bearer <token>`）
- 分页：`page` / `size`
- 返回结构：`{ code, message, data }`

## 认证
### 登录
- POST `/api/auth/login`
- Body：`{ username, password }`
- Response：`{ token, username, role }`

### 注册
- POST `/api/auth/register`
- Body：`{ username, password }`

## 公共接口（无需登录）
### 课程列表
- GET `/api/public/courses`

### 教练列表
- GET `/api/public/coaches`

### 排课列表
- GET `/api/public/schedules`

### 公告列表
- GET `/api/public/notices`

## 管理端（ADMIN）
### 用户管理
- GET `/api/admin/users`
- POST `/api/admin/users`
- PUT `/api/admin/users/{id}`
- DELETE `/api/admin/users/{id}`

### 教练管理
- GET `/api/admin/coaches`
- POST `/api/admin/coaches`
- PUT `/api/admin/coaches/{id}`
- DELETE `/api/admin/coaches/{id}`

### 课程管理
- GET `/api/admin/courses`
- POST `/api/admin/courses`
- PUT `/api/admin/courses/{id}`
- DELETE `/api/admin/courses/{id}`

### 排课管理
- GET `/api/admin/schedules`
- POST `/api/admin/schedules`
- PUT `/api/admin/schedules/{id}`
- DELETE `/api/admin/schedules/{id}`
- GET `/api/admin/schedules/{id}/coaches`
- PUT `/api/admin/schedules/{id}/coaches`（Body：`{ coachIds: [] }`）

### 预约管理
- GET `/api/admin/reservations`
- PUT `/api/admin/reservations/{id}/status?status=`

### 订单管理
- GET `/api/admin/orders`
- PUT `/api/admin/orders/{id}/pay-status?payStatus=`
- PUT `/api/admin/orders/{id}/refund`

### 评价管理
- GET `/api/admin/evaluations`
- DELETE `/api/admin/evaluations/{id}`

### 反馈管理
- GET `/api/admin/feedbacks`
- POST `/api/admin/feedbacks/{id}/reply`

### 系统参数
- GET `/api/admin/params`
- POST `/api/admin/params`
- PUT `/api/admin/params/{id}`

### 公告
- GET `/api/admin/notices`
- POST `/api/admin/notices`
- PUT `/api/admin/notices/{id}`
- DELETE `/api/admin/notices/{id}`

### 统计
- GET `/api/admin/stats/overview`

### 文件上传
- POST `/api/admin/files/upload`（multipart/form-data）
- 返回 `file_url` 可直接访问（`/uploads/**`）

## 用户端（USER）
### 预约
- GET `/api/user/reservations`
- POST `/api/user/reservations`
- PUT `/api/user/reservations/{id}/cancel`
  - 可选参数：`reason`

### 评价
- POST `/api/user/evaluations`

### 订单
- GET `/api/user/orders`

### 个人信息
- GET `/api/user/profile`
- PUT `/api/user/profile`

## 教练端（COACH）
### 我的排课
- GET `/api/coach/schedules`

### 我的预约
- GET `/api/coach/reservations`

### 预约状态更新
- PUT `/api/coach/reservations/{id}/status?status=`
