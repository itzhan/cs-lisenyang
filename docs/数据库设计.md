# 翔泳社管理系统数据库设计

## 说明
- 数据库：MySQL
- 软删字段：`deleted`（0正常 / 1删除）
- 审计字段：`create_time`、`update_time`

## 主要表

### sys_user（用户）
- 用途：系统登录账号（管理员/普通用户/教练）
- 关键字段：username、password、status、avatar、gender

### sys_role / sys_user_role（角色与用户角色）
- 用途：RBAC 角色与绑定
- 角色：ADMIN / USER / COACH

### coach_profile（教练档案）
- 用途：教练资质与介绍
- 关键字段：user_id、qualification、specialty、intro、avatar_url

### course（课程）
- 用途：课程基础信息
- 关键字段：course_name、course_type、level、duration_min、price、capacity、cover_url

### course_schedule（排课）
- 用途：课程时间段安排
- 关键字段：course_id、start_time、end_time、course_date、max_capacity、current_count、status
- coach_id：主教练（可空）

### course_schedule_coach（排课-教练关联）
- 用途：排课与多教练协同
- 关键字段：schedule_id、coach_id

### reservation（预约）
- 用途：用户预约记录
- 关键字段：schedule_id、user_id、status、cancel_time、cancel_reason

### orders（订单）
- 用途：收费/退款模拟
- 关键字段：order_no、reservation_id、amount、pay_status、refund_status、refund_time

### evaluation（课程评价）
- 用途：对课程的评价（不支持图片）
- 关键字段：course_id、reservation_id、score、content

### feedback（反馈）
- 用途：用户建议与意见

### notice（公告）
- 用途：公告发布

### sys_param（系统参数）
- 用途：系统配置项（如取消规则）
- 默认参数：
  - max_reserve_per_user
  - cancel_before_minutes
  - refund_enabled

### file_resource（文件）
- 用途：上传文件记录（课程封面/教练头像）
- 关键字段：file_name、file_path、file_url、uploader_id
